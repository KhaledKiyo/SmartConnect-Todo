# SmartConnect To-Do App 🚀

A modern, clean, and efficient To-Do application built with **Flutter** and **GetX**. 
This project is designed to help users manage their daily tasks with categories, priorities, and statistics tracking.

## ✨ Features
- **Task Management:** Create, edit, and delete tasks with ease.
- **Categorization:** Organize tasks by Personal, Work, Shopping, or Other.
- **Priority System:** Set tasks as Low, Medium, or High priority.
- **Statistics:** Track your productivity with an intuitive stats dashboard.
- **Theme Support:** Switch seamlessly between Light and Dark modes.
- **Data Persistence:** Tasks are saved locally using `GetStorage`.
- **Haptic Feedback:** Experience tactile feedback for interactions.

## 🛠 Tech Stack
- **Framework:** Flutter
- **State Management:** GetX
- **Local Storage:** GetStorage
- **UI:** Material Design 3

## 🚀 How to Run
1. Clone this repository:
   `git clone https://github.com/KhaledKiyo/SmartConnect-Todo`
2. Install dependencies:
   `flutter pub get`
3. Run the app:
   `flutter run`
