# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-12

### Added
- Initial release of Pro To-Do.
- Task management with categories (Personal, Work, Shopping, Other).
- Task prioritization (Low, Medium, High).
- Due date support with integrated date picker.
- Real-time search and live filtering.
- Statistics dashboard with progress tracking.
- Material 3 design with Dark/Light mode support.
- Local persistence using GetStorage.
- Haptic feedback for a tactile experience.
- Undo support for deleted tasks.
- Modular architecture for better maintainability.
